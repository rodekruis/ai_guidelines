# Checklist AI usage

## For the team

- [ ] every team member has read the following documents
  - [ ] [General Netherlands Red Cross guidelines on AI usage](https://kruispunt.rodekruis.nl/kenniscentrum/informatievoorziening-ict/ai-co-pilot?status=all)
  - [ ] [Responsible AI Guide](https://responsibleai.guide)
  - [ ] [How to use AI in this repository](how_to_use_ai.md)

## For a product (repository, project)

- [ ] the project repository states that it uses AI using a disclaimer like [this example](ai_disclaimer.md)
